## Features

* Folder definition
* Basic Git configuration
* Basic composer file 
* Travis CI configuration
* PHPUnit configuration
* License
* Contributing instruction
* Changelog template

## Install

```
composer create-project tm/new-php-project new-project --no-interaction
cd new-project
git init
cat .gitconfig > .git/config
```
